Documentation du plugin Terensys

[(#BOITE_OUVRIR)]

Présentation

Le plugin Terensys permet de gérer des objets éditoriaux personnalisés dans SPIP appelés « articles éditoriaux ».

Ces objets sont des blocs qui seront affichés en page d'accueil.

Ils peuvent être créés, édités, supprimés et triés par glisser-déposer depuis l'interface privée de SPIP.
Il seront affichés en page d'accueil selon leur ordre.

Divers objets éditoriaux sont associés à ces articles.

[(#BOITE_FERMER)] [(#BOITE_OUVRIR)]

Articles éditoriaux

Un article éditorial est initialement composé de :

  • Un titre
  • Un type

En fonction du type choisi, différents champs sont disponibles.

[(#BOITE_FERMER)] [(#BOITE_OUVRIR)]

Les types d'articles

    [(#BOITE_OUVRIR{'','sobre'})]

  • Texte avec image

    Les champs :

    • Texte
    • Document

    Affiche un article accompagné d'une image.

    • [(#BOITE_FERMER)] [(#BOITE_OUVRIR{'','sobre'})]

  • Texte avec image et action

    Les champs :

    • Texte
    • Document
    • Redirection
    • Label

    Affiche un article accompagné d'une image et d'un bouton redirigeant vers l'url de redirection.
    Le label permet de renseigner le texte à l'intérieur du bouton.

    • [(#BOITE_FERMER)] [(#BOITE_OUVRIR{'','sobre'})]

  • Texte

    Les champs :

    • Texte

    Affiche un article simple composé uniquement d'un titre et d'un texte.

    • [(#BOITE_FERMER)] [(#BOITE_OUVRIR{'','sobre'})]

  • Map

    Une fois créé, une carte vierge s'affiche. Il faut ensuite, depuis la page d'édition de l'article, ajouter des clients.
    Une fois les clients ajoutés, des points, situés aux adresses des clients, seront affichés sur la carte.

    • [(#BOITE_FERMER)] [(#BOITE_OUVRIR{'','sobre'})]

  • Carrousel

    Affiche un carrousel. Une fois créé, il faut, depuis la page d'édition de l'article, ajouter des clients.

    • [(#BOITE_FERMER)] [(#BOITE_OUVRIR{'','sobre'})]

  • 4 Blocs

    Les champs :

    • Articles

    Affiche 4 articles courts sous forme de 4 blocs disposés en carré. Il faut obligatoirement selectionner exactement 4 articles.

    • [(#BOITE_FERMER)] [(#BOITE_OUVRIR{'','sobre'})]

  • Partenaire

    Affiche les logos des partenaires avec leur description. Une fois créé, il faut, depuis la page d'édition de l'article, ajouter des partenaires.

    • [(#BOITE_FERMER)] [(#BOITE_OUVRIR{'','sobre'})]

  • Image avec cartes

    Les champs :

    • Sous titre
    • Texte
    • Document
    • Redirection
    • Label

    Affiche un article accompagné d'une image et de cartes de l'équipe à côté.

    Les champs ont les mêmes usages que ceux du Texte avec image et action

    • [(#BOITE_FERMER)]

Les Clients

Les Clients sont d'autres objets éditoriaux personnalisés.

Ils sont composés de :

  • Un nom
  • Une adresse
  • Un logo
  • Un article associé
  • une URL

L'adresse permet d'afficher le client sur la carte. Le logo peut être affiché dans le carrousel avec une redirection vers l'URL lors d'un clique sur celui-ci.

L'article associé est celui vers lequel on va aller lors d'un clique sur le client dans la page Références.

Les clients sont utilisés par la Carte, et la page Références.

[(#BOITE_FERMER)] [(#BOITE_OUVRIR)]

Les Partenaires

Les Partenaires sont d'autres objets éditoriaux personnalisés.

Ils sont composés de :

  • Un Nom
  • Une description
  • Un logo
  • Un article associé

Le tout va être utilisé par l'article éditorial Partenaire.

[(#BOITE_FERMER)] [(#BOITE_OUVRIR)]

Les Personnes

Les Personnes sont d'autres objets éditoriaux personnalisés.

Ils sont composés de :

  • Un Nom
  • Une fonction
  • Un texte
  • Une image

Le tout va être utilisé par l'article éditorial Image avec cartes et par la page Équipe.
